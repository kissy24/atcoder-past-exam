# atcoder-past-exam

Solve problems A~F of the Algorithm Practical Test Past Exams.

[Algorithm Practical Test Past Exams](https://atcoder.jp/contests/past201912-open/tasks)


## Usage

```sh
# example A problem
python ./src/A.py
```

## Requirements

Python 3.9

## Running Tests

To run all the tests, do:

```sh
pipenv run pytest src -vv
```
